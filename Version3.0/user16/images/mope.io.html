Dec 24: Bug Fixes & Optimizations


  • Another batch of server optimizations.
  • Small improvements to server networking code.
  • Should've fixed the server crash issues.
  • Should've fixed an issue with players accidentally having focus on an invisible ad (no keyboard input)
  • Fixed an issue with the player's bar type not being reset (no more lava drinking mouse)
  • Fixed Dino Monster and Giant Scorpion being invisible underwater.
  • Server physics are tick independent now, this means that something like 40 TPS will still feel close to 60 (smooth!)
  • Improved client rotation code to skip less frames, should be more smooth now.
  • Reverted fox ability behaviour (now should kick all animals below Rhino tier instead of only stunned animals inside holes).
  • Reduced the minimap pumpkins update time (now it updates every 1s)
  • Removed vertical rivers (currently they don't serve much purpose and take up a ton of server performance).
  • Removed expired LF servers from the list.
  • Fixed an issue with the shop animal increased rarity not working.
  • Fixed an issue where desert chipmunk spat food could've affected players inside arena.

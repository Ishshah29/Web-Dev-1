How to Create a List in HTML

Step-by-Step Guide

Follow these steps to create an unordered list in HTML:

1. Create a basic HTML structure

Start with the basic structure of an HTML document:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My List</title>
</head>
<body>
    <!-- Your content and list goes here -->
</body>
</html>

2. Add a heading for your list

Include a heading to introduce your list:

<h3>My Favorite Video Games</h3>

3. Create an unordered list

Use the <ul> tag to create an unordered list, and the <li> tags for each list item:

<ul>
    <li>The Witcher 3</li>
    <li>Titanfall 2</li>
    <li>Atomic Heart</li>
</ul>
